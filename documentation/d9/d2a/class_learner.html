Detailed Description

Holds a Program and an Action, using both to act on environment.

The Learner class contains a Program and an Action. When a Learner receives a feature set, it evaluates the feature set using its stored Program. The Program produces a bid, which the Learner's host team compares against all other Learners under its control. The Learner with the highest bid will then return its Action to the Team, which will be enacted on the environment.
