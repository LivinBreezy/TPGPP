Detailed Description

Holds a matrix of memory for Learners to access.

The MemoryModel class contains a single matrix which represents the global memory state of a TPG learning environment. It is generated by providing the row and column dimensions of the model, or by providing an existing MemoryModel to copy from. The matrix is created and the storage probabilities are generated based on the generateProbabilities() function. Once the probabilities are created, the memory is available for reading and writing.
