Detailed Description

Represents a register-based instruction similar to assembly.

The Instruction class contains a variety of information as it pertains to register-based instructions. It holds a mode value (for determining the type of source to read from), a source index value (for determining which index of the given source structure to read from), and a destination index value (for determining the other value to operate on as well as where to store the result). The class also holds a function which represents the functionality of the defined operation.
