Detailed Description

Base abstract class used for defining a reproductive class.

The Reproduction class is a fully abstract (pure virtual) class which defines the four required methods for creating a reproductive class in TPG. The methods are designed to be used in tandem as to:

  1. Use teamSelection to return a list of the best ranked Teams.
  2. Use teamReproduction to produce the children of the best ranked Teams.
  3. Use teamMutation to mutate those children.
  4. Use teamUnify to combine the best parents and their children into a single Team population.
