Detailed Description

The main class for the TPG learning process.

The TpgAlgorithm class begins by accepting a file path to a parameters file and a label for the type of algorithm to run. Once it knows whether to start a learning process or testing process, it reads in the parameters file and stores it in a map, then creates either a TpgLearn or TpgPlay object and provides parameters so they can process.
